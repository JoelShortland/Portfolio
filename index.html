GUI Java Application with API Consumption

This GUI application, built with JavaFX, allows for the user to navigate Marvel API data. It sends and retrieves information via HTTP requests, allowing for the retreval of Marvel data and the export of data to Reddit. Additionally, the program utilises SQLite to cache data to minimise API calls. Developed using a Model-View-Controller (MVC) framework, it follows a structured and maintainable design.
